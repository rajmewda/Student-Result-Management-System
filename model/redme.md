# Student Result Management System

## Description
A Java-based **Student Result Management System** built using **Object-Oriented Programming (OOP)** concepts.  
The system allows adding students, storing marks, calculating total marks, percentage, and grade.  
It also supports searching students by roll number and displaying all student results.  

---

## Features
- Add student records with ID, Name, Roll Number, and marks.  
- Calculate **Total Marks**, **Percentage**, and **Grade** automatically.  
- Search a student by **Roll Number**.  
- Display results for all students.  
- Uses **OOP concepts**: inheritance, encapsulation, modular design.  

---

## Technologies
- Java 8+  
- Console-based application  
- Multi-package project structure
